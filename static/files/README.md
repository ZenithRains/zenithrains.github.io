# Downloadable files

Place the public CV and other downloadable PDFs in this directory. For example,
`static/files/cv-ruiyi-zhang.pdf` will be published at
`/files/cv-ruiyi-zhang.pdf`.

The v1 public release of the PDE energy-estimate note is stored as
`static/files/pde-energy-categorical-semantics-v1.pdf` and published at
`/files/pde-energy-categorical-semantics-v1.pdf`.
